# Wiki is an OKF-conformant bundle, maintained via the LLM Wiki process

Every page the agent writes is a valid OKF v0.1 concept document (frontmatter with `type`, conventional `index.md`/`log.md`, bundle-relative links) — not just a wiki loosely inspired by the pattern.
We chose this over treating LLM Wiki and OKF as two separate, unconnected demos because OKF explicitly cites LLM-wiki repositories as a related pattern (OKF §10), and its `index.md`/`log.md` conventions map directly onto LLM Wiki's own indexing/logging sections. One demo satisfying both specs at once is simpler than building two.
