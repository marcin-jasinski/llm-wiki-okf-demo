# No Obsidian integration

The LLM Wiki spec leans heavily on Obsidian as its companion app (Web Clipper for sourcing, graph view, Dataview, hotkey-driven asset download — "Obsidian is the IDE; the LLM is the programmer"), but this demo deliberately doesn't integrate with it. The wiki is plain OKF markdown, viewable in any editor; requiring Obsidian would add a dependency the audience has to have installed and configured for the demo to make sense, for a tool this app doesn't otherwise need. May be revisited later if there's time to show the graph-view payoff live, but it's not in scope now.
